## 补充说明

- 不推荐前端初学者直接看面经，建议先系统学习基础知识，参考 [学习资源](/resources/)。
- 在学习面经的过程中，还可以结合 [笔试相关](/exam/) 和 [面试相关](/interview/) 进行学习。
- 如遇错误、错别字，或有需要补充的知识点，请先阅读 [贡献指南](/about/contribution/)。
